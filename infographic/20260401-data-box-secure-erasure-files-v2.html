Azure Data Box: セキュア消去証明書の自動生成と Azure Files Provisioned v2 対応

NIST 800-88 Revision 2 準拠のセキュア消去証明書が自動生成されるようになり、コンプライアンス対応が大幅に簡素化。さらに Azure Files Provisioned v2 ストレージアカウントへの直接データ取り込みが可能になり、移行先の柔軟性が向上しました。

🔄 概要: Before / After 比較

Azure Data Box に 2 つの機能強化が同時に一般提供 (GA) されました。セキュア消去証明書の自動生成により監査対応が自動化され、Azure Files Provisioned v2 対応により移行先ストレージの柔軟性が向上しました。

Before

  • データ消去のコンプライアンス証跡を手動で管理・取得する必要があった
  • セキュア消去の証明は注文履歴のログのみで、正式な証明書としてダウンロード不可
  • Provisioned v2 課金モデルのストレージアカウントを送信先として選択できなかった
  • Provisioned v1 ではパフォーマンス設定がサイズに連動し、過剰プロビジョニングが発生

After

  • 注文完了時にセキュア消去証明書が自動生成され、Azure ポータルから直接ダウンロード可能
  • NIST 800-88r2 準拠の消去方法・検証手法・デバイス情報を含む証明書で監査対応を効率化
  • Azure Files Provisioned v2 ストレージアカウントを送信先として選択可能に
  • 容量・IOPS・スループットを個別に設定してコスト最適化が可能

✨ 主要機能
📜

セキュア消去証明書の自動生成

Data Box の注文完了時に NIST 800-88 Revision 2 準拠の証明書が自動生成されます。デバイスモデル、シリアル番号、ディスク情報、消去方法、検証手法、実施日時が記載され、Azure ポータルからダウンロード可能です。追加の操作は不要です。

🛡️

NIST 800-88r2 準拠のデータ消去

Data Box 120 TB / 525 TB では ARCCONF 4.17.00 ツール、Data Box Disk では MSECLI ツールによるブロック消去 (Purge レベル) を実施。ランダム 10% サンプリング + 二次 2% サンプリングによる検証で消去の完全性を保証します。

📁

Azure Files Provisioned v2 へのデータ取り込み対応

Data Box の送信先として Provisioned v2 課金モデルの Azure Files ストレージアカウントを選択可能に。容量・IOPS・スループットを個別にプロビジョニングできる柔軟な課金モデルにより、ワークロードに応じたコスト最適化を実現します。

⚖️ メリット / 制約事項

メリット

ビジネス面

  • 監査対応に必要な証跡の取得が自動化 (HIPAA, GDPR, 金融規制対応)
  • 証明書の手動取得・管理の工数が不要に
  • Provisioned v2 で過剰プロビジョニングを回避しコスト最適化

技術面

  • 最新の NIST 800-88r2 標準に準拠
  • 容量・IOPS・スループットを個別にプロビジョニング可能
  • 移行後のストレージ再構成が不要

制約事項

  • 証明書はデバイス返却・消去完了後にのみ利用可能
  • Provisioned v2 は FileStorage 種類のストレージアカウントが必要
  • コマース境界を超えた配送は非対応
  • OS ディスクの移行は非対応
  • Provisioned v2 の新リソースプロバイダー (Microsoft.FileShares) は現時点でプレビュー (NFS のみ)

規制産業でのデータセンター移行

金融機関や医療機関がオンプレミスのファイルサーバーを Azure に移行する際、Data Box で大量データを転送し、セキュア消去証明書で監査要件を満たします。

🎬

メディア・コンテンツ管理

大量の映像・画像データを Azure Files に移行し、Provisioned v2 でストリーミングに必要な IOPS とスループットを個別に確保します。

💾

バックアップ・アーカイブ移行

オンプレミスのバックアップデータを Azure Files に移行し、コンプライアンスに必要な消去証明を自動取得します。

☁️

ハイブリッドクラウド環境の構築

Azure File Sync と組み合わせて大量の初期データを Data Box で移行し、以降は差分を同期するハイブリッド構成を実現します。
