概要

Azure Cosmos DB Shell がパブリックプレビューとして公開。オープンソースのCLIで、bashライクなコマンド構文を使用してCosmos DBと直感的に対話できる。最大の特徴はModel Context Protocol (MCP) サーバーサポートによるAIエージェント連携機能。

Key Points

  • bash ライクなコマンド構文でデータベース操作
  • MCP サーバーによる AI エージェント連携
  • パイプ・スクリプティングによる自動化
  • VS Code 拡張機能との統合
  • MIT ライセンスで無料利用可能

Before

  • Cosmos DBのデータ操作にはAzure Portal、SDK、またはData Explorerが必要で、コマンドラインからの迅速な操作が困難
  • AIアシスタントがCosmos DBに直接アクセスする標準的な手段がなかった
  • スクリプト化にはSDKを使用したカスタムコードが必要

After

  • bashライクなコマンドで直感的にデータベースを操作可能
  • MCPサーバーを通じてAIアシスタントがCosmos DBと直接連携可能
  • パイプやスクリプティングにより複雑なワークフローの自動化が実現
  • VS Code拡張機能によりエディタからシームレスに利用可能

Installation

# NuGet (推奨)
dotnet tool install --global CosmosDBShell --prerelease

# MCP サーバー起動
cosmosdbshell --mcp --mcp-port 6128

MCP Configuration

{
  "mcpServers": {
    "cosmosdb": {
      "command": "cosmosdbshell",
      "args": ["--mcp"],
      "env": { "MCP_PORT": "6128" }
    }
  }
}

Use Cases

UC1: 開発・テスト

コマンドラインからの迅速なデータ確認・操作。bash ライクなコマンドでデータベースの内容を素早く確認し、テストデータの挿入・削除を効率化。

UC2: AI 連携

MCP を通じた AI アシスタントによるデータ操作・分析。Claude や ChatGPT が Cosmos DB のデータに直接アクセスし、ユーザーの自然言語指示に基づいてクエリを実行。

UC3: 自動化

シェルスクリプトを使用した定型作業の自動化。パイプによるデータ変換、定期的なデータエクスポート、バッチ処理などを CLI スクリプトで実現。

Pricing

無料 (MIT ライセンス)

Cosmos DB Shell 自体は無料で利用可能。クエリ・操作に対する Cosmos DB 利用料金 (RU/s) は通常通り課金されます。

Limitations / Caveats

  • プレビュー段階、本番環境での使用は推奨されない
  • NuGet インストールには .NET SDK 10.0 以降が必要
  • MCP サーバーは localhost バインド
  • 自己完結型バイナリは約 200MB
