Microsoft Build 2026 で発表された 5 つの音声・アバター機能アップデート

GA (一般提供) In Preview 2026-06-02 ~ 2026-06-04

概要

Microsoft Build 2026 において、Microsoft Foundry の音声およびアバター機能に関する 5 つのアップデートが発表された。Voice Live API と Foundry Agent Service の統合が GA となり、開発者は独自のオーディオパイプラインを構築することなく、リアルタイム音声エージェントを構築できるようになった。加えて、Microsoft AI チームによるファーストパーティ音声モデル MAI-Voice-2 がパブリックプレビュー、アバター関連では Voice Live API でのカスタムボイスを使ったアバター音声同期がプレビュー開始、セルフサーブ型のカスタムフォトアバター作成とカスタムボイスポータルが GA となった。

5 つの主要アップデート

  • Voice Live + Foundry Agent Service 統合 (GA) - WebSocket ベースのリアルタイム音声エージェント構築
  • MAI-Voice-2 (プレビュー) - Microsoft AI チーム開発のファーストパーティ音声モデル
  • Voice Live API アバター音声同期 (プレビュー) - カスタムボイスとリアルタイムアバターの組み合わせ
  • セルフサーブ カスタムフォトアバター (GA) - 1 枚の写真から VASA-1 ベースのアバター作成
  • Custom Voice ポータル体験 (GA) - Foundry ポータルに統合された一元管理

Before

  • リアルタイム音声エージェント構築には複雑なオーケストレーションが必要だった
  • カスタムボイスの作成・管理は Speech Studio で別ワークフローだった
  • カスタムフォトアバターの作成にはオフラインプロセスが必要だった
  • Voice Live API でカスタムボイスとアバターの音声同期を組み合わせられなかった

After

  • Voice Live API が Foundry Agent Service と直接統合、単一インターフェースで構築可能
  • カスタムボイスが Microsoft Foundry ポータルに統合
  • セルフサーブでカスタムフォトアバター作成可能
  • Voice Live API でカスタムボイスとアバターの音声同期がサポート
  • MAI-Voice-2 により高品質音声モデルが利用可能

Use Cases

UC1: ブランド専用音声エージェント (コンタクトセンター)

企業独自のブランドボイスを持つ音声エージェントにカスタムフォトアバターを組み合わせ、視覚的にも一貫したブランド体験をコンタクトセンターで提供。Function Calling で既存の業務システムと連携し、問い合わせ対応を自動化する。

構成: Voice Live API + Custom Voice + Custom Photo Avatar + Function Calling

UC2: 多言語対応教育アシスタント

MAI-Voice-2 の多言語対応と GPT-5 の知識を組み合わせた教育アシスタント。10+ 言語で自然な音声対話を実現し、学習者の母国語で個別指導を提供する。

構成: Voice Live API + MAI-Voice-2 + GPT-5 (10+ 言語対応)

UC3: 企業内 HR バーチャルアシスタント

カスタムフォトアバターを使った親しみやすい HR アシスタント。Basic ティアのモデルでコスト最適化しつつ、従業員からの人事関連の問い合わせに音声で対応する。

構成: Voice Live API (Basic tier) + Custom Photo Avatar
