Azure Monitor Launched (GA)

Dashboards with Grafana が Public、Government、China で一般提供開始

リリース日: 2026-05-12

📋概要

Azure Monitor dashboards with Grafana が一般提供 (GA) となり、Public クラウド、Government (Fairfax)、China リージョンで利用可能に。Grafana のオープンかつコンポーザブルな可視化プラットフォームが Azure Portal 内に直接統合され、追加コストなし・設定不要で利用できる。

❌ Before

  • 可視化は Azure Monitor ネイティブダッシュボード/ワークブックに限定
  • Grafana には別途プロビジョニングと追加コストが必要
  • Government/China では Grafana 可視化オプションが限定的
  • コミュニティダッシュボード活用に外部インスタンスが必要

✅ After

  • Azure Portal 内で Grafana ダッシュボードを作成・編集・共有
  • 追加コストなし、設定不要で即座に利用開始
  • 数千のコミュニティダッシュボードをインポート可能
  • Azure RBAC / ARM/Bicep テンプレートによる自動化対応
  • 全クラウド環境 (Public, Government, China) で利用可能

Azure Portal 内で動作し、複数の Azure データソースに接続してデータを可視化。必要に応じて Azure Managed Grafana へダッシュボードをコピーして高度な機能を利用可能。

⚡主要機能

📊

プリビルトダッシュボード

Azure リソースのモニタリングデータを可視化するための事前構築済みダッシュボードが提供。Kubernetes クラスターの Prometheus メトリクス可視化にも対応。

🌐

Grafana コミュニティダッシュボードのインポート

数千の公開 Grafana コミュニティダッシュボードをインポートして利用可能。Prometheus データソース対応ダッシュボードをそのまま活用。

🎨

カスタムダッシュボードの作成

Azure Portal 内で独自のダッシュボードを作成可能。Grafana の豊富なビジュアライゼーションとクライアントサイドのデータ変換機能を利用。

🔐

Azure ネイティブリソース管理

Grafana ダッシュボードを Azure ネイティブリソースとして管理。Azure RBAC によるアクセス制御、ARM/Bicep テンプレートによる自動化に対応。

📤

Azure Managed Grafana へのコピー

作成したダッシュボードを Azure Managed Grafana インスタンスにコピー可能。パネル、クエリ、レイアウトが保持される。

🔧技術仕様

💡ユースケース

🐳 Kubernetes クラスターのモニタリング

AKS クラスターの Prometheus メトリクスを Grafana ダッシュボードで可視化し、チーム全体で共有する。Azure Monitor で Prometheus スクレイピングを有効化し、プリビルトまたはコミュニティダッシュボードを活用。

✨ 追加コストなしで Kubernetes クラスターの包括的モニタリングダッシュボードを構築・共有

📈 マルチリソースの統合監視ダッシュボード

複数の Azure リソース (VM、ストレージ、データベース) のメトリクスとログを単一のダッシュボードで統合監視。Azure Monitor Metrics と Azure Monitor Logs (KQL) を組み合わせてパネルを構成。

✨ 単一ビューでインフラ全体の健全性を把握し、問題の早期検知とトラブルシューティングを迅速化

⚠️制約事項

  • Grafana アラート機能は非対応 (アラート必要時は Azure Managed Grafana を使用)
  • レポートのスケジュール機能は非対応
  • ライブラリパネル、スナップショット、プレイリスト、App プラグインは非対応
  • プライベートネットワーク (Private Link) は非対応
  • 外部データソース (OSS/エンタープライズデータソース) は非対応
  • 認証は現在のユーザー ID のみ (マネージド ID/アプリ登録は不可)
  • Grafana Enterprise 機能は非対応
