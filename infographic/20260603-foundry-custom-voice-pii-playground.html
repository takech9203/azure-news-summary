  • すべての AI サービスを単一ポータルで管理・テスト

⭐ 主要機能

GA Custom Voice ポータル体験

🎤

ボイスタレント管理

同意声明のアップロード、Limited Access による承認制アクセス管理。安全なカスタム音声作成の基盤。

📊

データ準備と品質チェック

信号対雑音比 (SNR) の測定、発話の一貫性チェック。最低 300 発話以上のトレーニングデータ要件。

🧠

モデルトレーニング

ニューラル TTS エンジンによる高品質音声合成。多言語対応でグローバル展開が可能。

🌐

エンドポイント管理

REST API、Speech SDK、SSML による柔軟なデプロイメントと制御。

Preview Conversational PII NextGen Playground

💬

マルチスピーカー対話形式入力

複数話者の会話トランスクリプトを入力して PII 検出をテスト。コールセンターシナリオに最適。

⚙

API Configuration Panel

API バージョン、パラメータ設定を GUI で直感的に操作。コード不要でテスト可能。

🔍

検出結果可視化

エンティティカテゴリと信頼度スコアの可視化。検出精度の即座確認。

⏳

非同期ジョブ処理テスト

大量データの非同期処理をプレイグラウンドから実行・モニタリング。

💡 ユースケース

1

ブランドボイスによる顧客対応自動化

Custom Voice で独自ブランドの音声を作成し、IVR (自動音声応答) システムに統合。一貫したブランド体験を顧客に提供。

2

コールセンタートランスクリプトの PII 除去

Conversational PII を活用し、会話記録から個人情報を自動検出・墨消し。GDPR やプライバシー規制への対応を効率化。

3

アプリケーションログの PII 事前検証

TextPII Playground で SaaS アプリケーションのログ出力を検証。個人情報の意図しない記録を事前に防止し、ログ監査を効率化。
