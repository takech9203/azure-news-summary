SQL Server on Azure Virtual Machines

Malaysia West / Indonesia Central リージョン展開

SQL Server on Azure VMs が東南アジア 2 リージョンで一般提供開始

GA (一般提供) 2026-05-20 Database / Compute

概要

SQL Server on Azure Virtual Machines が Malaysia West および Indonesia Central リージョンで一般提供 (GA) となりました。これにより、マレーシアおよびインドネシアのユーザーは、データ主権要件を満たしながら低レイテンシで SQL Server ワークロードを実行できるようになります。SQL Server 2016 から 2025 までの全バージョン、Windows Server および Linux をサポートし、従量課金制と Azure Hybrid Benefit (BYOL) の両方のライセンスモデルに対応しています。

Key Points

  • Malaysia West および Indonesia Central の 2 つの新しい東南アジアリージョンで GA
  • SQL Server 2016, 2017, 2019, 2022, 2025 をサポート
  • Windows Server 2016-2025 および Linux に対応
  • Pay-as-you-go と Azure Hybrid Benefit (BYOL) の 2 つのライセンスモデル
  • SQL IaaS Agent Extension (無料) による自動管理機能を提供

Before

  • マレーシア/インドネシアのユーザーは地理的に離れたリージョンにデプロイする必要があった
  • データ主権 (Data Residency) 要件を満たせない場合があった
  • 距離に起因する高レイテンシが業務に影響

After

  • Malaysia West / Indonesia Central に直接デプロイ可能
  • データ主権コンプライアンスを達成
  • ローカルリージョンによるレイテンシ大幅改善

Malaysia West

マレーシア西部リージョン。マレーシア国内のデータ主権要件に対応し、金融機関や政府機関のワークロードに最適。

主要機能 (SQL IaaS Agent Extension)

💻

Azure Portal 管理

SQL Server VM の一元管理。Azure Portal から SQL Server インスタンスの状態監視、構成変更、診断情報の確認を統合的に行えます。

📦

自動バックアップ

すべてのデータベースに対するバックアップスケジューリングの自動化。ポイントインタイムリストアに対応した包括的なバックアップ戦略を簡単に構成できます。

🔄

自動パッチ適用

Azure Update Manager を通じたセキュリティパッチおよび重要な更新プログラムの自動インストール。メンテナンスウィンドウの指定も可能です。

🔒

Azure Key Vault 統合

暗号化キーや証明書の自動インストールおよび構成。TDE (Transparent Data Encryption) との統合によりデータ保護を強化します。

📊

SQL ベストプラクティス評価

SQL Server のヘルスチェックと最適化推奨事項の提供。パフォーマンス、セキュリティ、可用性に関するベストプラクティスへの準拠度を評価します。

🔐

Microsoft Entra 認証

Azure Active Directory (Entra ID) による認証をサポート。パスワードレス認証やゼロトラストセキュリティモデルの実現に貢献します。

🛡️

拡張セキュリティ更新 (ESU)

サポート終了後の SQL Server に対して最大 3 年間のセキュリティ更新を提供。レガシーワークロードの保護を継続しながら移行計画を策定できます。

ユースケース

🏦

マレーシアの金融機関向けデータベース移行

マレーシア国内のデータ主権要件を満たしながら、オンプレミスの SQL Server を Azure に移行。Malaysia West リージョンにデプロイすることで、Bank Negara Malaysia (BNM) の規制要件に準拠しつつクラウドのスケーラビリティを活用できます。

🛒

インドネシアの EC サイト向けバックエンド

Indonesia Central リージョンへのデプロイにより、インドネシア国内ユーザーへのレイテンシを大幅に削減。ピーク時のトランザクション処理性能を向上させ、ユーザー体験の改善とコンバージョン率の向上を実現します。

🛡️

東南アジア DR 構成 (Always On 可用性グループ)

Malaysia West と Indonesia Central を組み合わせた東南アジア内でのディザスタリカバリ構成。Always On 可用性グループにより、リージョン障害時にも短い RTO/RPO でビジネス継続性を確保します。

ライセンスモデル

Pay-as-you-go

SQL Server ライセンスを従量課金制で利用。初期投資不要で、使用した分だけ支払います。開発/テスト環境や短期プロジェクトに最適。

VM コスト + SQL Server ライセンス (時間課金)

Azure Hybrid Benefit (BYOL)

既存の SQL Server ライセンス (ソフトウェア アシュアランス付き) を Azure に持ち込み。ライセンスコストを最大 55% 削減可能。

VM コストのみ (既存ライセンス適用)
