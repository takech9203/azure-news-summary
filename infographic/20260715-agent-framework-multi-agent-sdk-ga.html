📋 概要

Microsoft Agent Framework が正式リリース (GA) を迎えた。AutoGen と Semantic Kernel のパターンを統合した統一 C#/Python SDK を中心に、プロダクション対応のエージェントランタイム「Agent Harness」、GAIA ベンチマークで 38% を達成するマルチエージェントオーケストレーションパターン「Magentic」、GitHub Copilot および Claude Code へのコネクタの 4 つの主要機能が一般提供開始となった。

⭐ 主要機能

🔗

Multi-Agent Orchestration SDK (C#/Python)

AutoGen と Semantic Kernel を統合した統一抽象レイヤー。グラフベースのワークフロー (WorkflowBuilder) と関数型ワークフロー API (Python デコレータ) の 2 つのアプローチでマルチエージェントオーケストレーションを構築可能。

🛡

Agent Harness

プロダクション対応のエージェントランタイム。計画 (Planning)、ToDo 管理、コンテキスト圧縮 (Context Compaction)、ツール自動承認 (Tool Auto-Approval)、OpenTelemetry による可観測性を備え、大規模言語モデルの制約内で安定動作を実現。

🧲

Magentic パターン

マルチエージェントオーケストレーションパターン。GAIA ベンチマークで 38% を達成し、AssistantBench および WebArena でも競争力のあるスコアを記録。複雑なタスク分解と協調実行を実現する。

🔌

GitHub Copilot / Claude Code コネクタ

カスタムアダプタなしで GitHub Copilot および Claude Code にコーディングタスクを委任可能。エージェントワークフロー内からシームレスにコード生成・レビューを実行できる。

💻 コード例

python Agent Harness を使用したエージェント作成 
from agent_framework import create_harness_agent
from agent_framework.openai import OpenAIChatClient
from azure.identity import AzureCliCredential

client = FoundryChatClient(
    project_endpoint="https://your-foundry-service.services.ai.azure.com/api/projects/your-project",
    model="gpt-5.4-mini",
    credential=AzureCliCredential(),
)

agent = create_harness_agent(
    client=client,
    max_context_window_tokens=128_000,
    max_output_tokens=16_384,
)

💡 ユースケース

💻 マルチエージェントコード生成パイプライン

Analyst (要件分析) → Coder (GitHub Copilot でコード生成) → Reviewer (レビュー) の 3 エージェント構成を WorkflowBuilder で構築。各エージェントが専門タスクを担当し、協調してコード生成を完了する。

WorkflowBuilder のグラフベースワークフローで Sequential/Concurrent パターンを組み合わせ、Copilot コネクタによりコーディング部分を自動委任

🔍 自律型リサーチエージェント

Agent Harness の Plan/Execute モードを活用し、複雑な調査タスクを自動的にサブタスクに分解。ToDo 管理でタスク進捗を追跡し、コンテキスト圧縮で長期実行を安定化する。

Agent Harness の Planning + ToDo 管理 + Context Compaction により、大量の情報を扱うリサーチタスクでもコンテキストウィンドウを超えない安定動作を実現
