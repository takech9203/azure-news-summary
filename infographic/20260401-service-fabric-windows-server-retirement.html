Azure Service Fabric Retirement

Windows Server 2019/2022 サポート終了 - Windows Server 2025 へのアップグレードが必要

リリース日: 2026-04-01 | Azure Service Fabric | Compute
📋 概要

Microsoft は、Azure Service Fabric における Windows Server 2019 および Windows Server 2022 上で稼働するクラスターのサポートを 2027 年 3 月 31 日 に終了します。全てのクラスターを Windows Server 2025 へアップグレードする必要があります。WS2025 のサポートは 2029 年 11 月 13 日まで継続されます。

本リタイアは Windows Server 自体のライフサイクルに基づくものです。OS のサポート終了に伴い、Service Fabric クラスターのサポートも連動して終了します。

🔑 主要な変更点
⚠

Windows Server 2019 サポート終了

Windows Server 2019 上で稼働する Service Fabric クラスターのサポートが 2027 年 3 月 31 日に終了します。以降はセキュリティ更新やバグ修正が提供されなくなります。

⚠

Windows Server 2022 サポート終了

Windows Server 2022 上で稼働する Service Fabric クラスターのサポートも同日 (2027 年 3 月 31 日) に終了します。

✅

Windows Server 2025 への移行が必須

移行先の Windows Server 2025 は 2029 年 11 月 13 日までサポートされます。Service Fabric ランタイム 11.x (11 CU4) が利用可能です。

⚖ メリットと制約事項

メリット

  • 長期サポート (2029 年 11 月まで)
  • セキュリティ強化
  • 最新機能の活用 (Service Fabric 11.x)
  • パフォーマンス改善

デメリット・制約事項

  • OS アップグレードにはダウンタイム計画が必要
  • WS2025 との互換性検証が必要
  • 大規模クラスターでは各ノード個別アップグレードで作業量増大
  • 移行猶予期間 約 12 か月
💻 対応方法 - コードサンプル

スタンドアロンクラスターの場合

PowerShell 
# クラスターに接続
Connect-ServiceFabricCluster -ConnectionEndpoint "yourcluster:19000"

# 現在のクラスターバージョンを確認
Get-ServiceFabricClusterUpgrade

# アップグレード可能なバージョンを一覧表示
Get-ServiceFabricRegisteredClusterCodeVersion

# Service Fabric ランタイムをアップグレード
Start-ServiceFabricClusterUpgrade -Code -CodePackageVersion <version> -Monitored -FailureAction Rollback

# アップグレードの進捗を確認
Get-ServiceFabricClusterUpgrade

自動アップグレード設定

JSON 
{
    "fabricClusterAutoupgradeEnabled": true
}
💡 ユースケース

1. Azure マネージドクラスター

Azure Portal または ARM テンプレートで OS SKU を Windows Server 2025 に変更します。ローリングアップグレードが自動管理されるため、最小限のダウンタイムで移行が可能です。

2. オンプレミスのスタンドアロンクラスター

テスト環境で互換性検証を実施後、本番環境でノードごとのローリングアップグレードを実行します。Service Fabric ランタイムも最新版に更新することを推奨します。
