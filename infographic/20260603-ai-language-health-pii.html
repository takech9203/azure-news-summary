臨床テキスト解析の本番利用開始と PII 検出プレイグラウンドの大幅拡充

GA Public Preview 2026-06-03

概要

Microsoft Build 2026 において、Azure AI Language の NextGen Playground に関する 3 つのアップデートが発表された。Text Analytics for Health の GA、Conversational PII プレイグラウンドのプレビュー、TextPII プレイグラウンドのアップデートにより、臨床データ解析とプライバシー保護の検証体験が大幅に向上。

Key Points

  • Text Analytics for Health (TA4H) NextGen Playground が一般提供開始 - 臨床テキストから構造化医療エンティティを即座に抽出
  • Conversational PII NextGen Playground がパブリックプレビュー - マルチスピーカー対話データの PII 検出を検証
  • TextPII NextGen Playground がアップデート - API Configuration Panel 刷新、リダクションモードを直接テスト
  • すべてのプレイグラウンドが Microsoft Foundry ポータルに統合

Before

  • TA4H のプレイグラウンドはプレビュー段階で SLA 保証なし
  • PII 検出の検証には API を直接呼び出す必要があり、インタラクティブな UI が限定的
  • マルチスピーカーの会話データに対する PII 検出を統合前に試す手段が不足
  • TextPII のプレビュー機能をプレイグラウンドで試せなかった

After

  • TA4H NextGen Playground が GA となり本番利用可能
  • Conversational PII 用の専用プレイグラウンドでマルチスピーカー対話の PII 検出を視覚的に検証可能
  • TextPII プレイグラウンドで API Configuration Panel からリダクションモードやエンティティフィルタをインタラクティブに設定
  • すべてのプレイグラウンドが Microsoft Foundry ポータルに統合

TA4H NextGen Playground (GA)

臨床テキストから構造化された医療エンティティを即座に抽出し、FHIR 準拠で出力。

  • NER、Relation Extraction、Entity Linking、Assertion Detection
  • FHIR 出力対応で電子カルテとの連携が容易
  • Social Determinants of Health (SDOH) 対応
  • 7 言語対応 (英語、ドイツ語、フランス語、イタリア語、スペイン語、ポルトガル語、ヘブライ語)
🔒

Conversational PII (Preview)

マルチターン会話構造に対応した PII 検出をプレイグラウンドで視覚的に検証。

  • マルチターン会話構造対応
  • スピーカー別ターン分離を維持した PII 検出
  • 非同期ジョブ処理でバッチ分析にも対応
🛡

TextPII (Preview)

刷新された API Configuration Panel でリダクションモードやエンティティフィルタをインタラクティブに設定。

  • 刷新された API Configuration Panel
  • 定義済み PII カテゴリの選択とテスト
  • リダクションモードの設定と即時プレビュー
  • 同期型リクエスト/レスポンス統合
⚙

Microsoft Foundry ポータル統合

すべての NextGen Playground が Microsoft Foundry ポータルに統合され、統一的な操作体験を提供。

  • 統一された UI/UX で全プレイグラウンドにアクセス
  • API 統合前のインタラクティブな検証ワークフロー

ユースケース

臨床データの構造化

退院サマリーから診断名、投薬情報、バイタルサインを自動抽出し FHIR 準拠で電子カルテ連携

  • TA4H NextGen Playground で臨床テキストの解析精度を事前検証
  • FHIR 出力を活用して電子カルテシステムとシームレスに統合
  • SDOH 対応で社会的決定要因の情報も構造化

コールセンター会話のプライバシー保護

通話録音のトランスクリプトから PII を自動リダクトし、コンプライアンス要件を遵守

  • Conversational PII Playground でマルチスピーカー対話の PII 検出を検証
  • スピーカー別ターン分離を維持したまま個人情報をマスク
  • 非同期処理で大量の通話ログをバッチ処理

AI ワークフローのプロンプトフィルタリング

LLM に送信するプロンプト/レスポンスの PII をリアルタイムリダクト

  • TextPII Playground でリダクションモードとエンティティフィルタの設定を事前テスト
  • 同期型 API で LLM パイプラインにインラインで組み込み
  • 定義済み PII カテゴリで検出対象を柔軟にカスタマイズ
