Azure Policy による Model Router デプロイメントの一元的なガバナンスとコンプライアンス管理

Preview 2026-06-02

概要

Azure Policy が Microsoft Foundry Models の Model Router デプロイメントに対する一元的なガバナンスを提供開始。組織はモデル選択のルーティング標準を定義・強制し、セキュリティ、コンプライアンス、運用要件を確実に満たすことが可能になった。

Key Points

  • Model Router デプロイメントのルーティング設定に対するポリシー定義が可能
  • 管理グループ、サブスクリプション、リソースグループ スコープでの一元的なポリシー割り当て
  • デプロイ時および 24 時間サイクルでの自動コンプライアンス評価
  • Deny, Audit, AuditIfNotExists の複数ポリシー効果をサポート

Before

  • Model Router のルーティング標準に対する一元的なガバナンスが存在しない
  • コンプライアンス検証が自動化されておらず手動確認が必要
  • ポリシーチェックが手動で行われ、スケールに対応困難

After

  • Azure Policy による組織全体のルーティング標準の自動強制
  • デプロイ時と定期サイクルでの自動コンプライアンス評価
  • 管理グループレベルでの一元的ガバナンスをスケールで実現

Model Router 向けポリシー定義

Model Router デプロイメントのルーティング構成に対するポリシールールを定義。特定のデプロイメントタイプや SKU に基づいた制御が可能。

🔗

一元的なルーティング標準の強制

管理グループ、サブスクリプション、リソースグループの各スコープでポリシーを割り当て、組織全体で統一したルーティング標準を強制。

📊

コンプライアンス評価と監査

デプロイ時にリアルタイムで評価を実施し、さらに 24 時間サイクルで定期的にコンプライアンス状態を再評価。非準拠リソースを自動検出。

🛡

複数のポリシー効果のサポート

Deny (拒否)、Audit (監査)、AuditIfNotExists (存在しない場合監査) など、要件に応じた柔軟なポリシー効果を選択可能。

ポリシー定義サンプル

GlobalStandard SKU のデプロイメントを拒否するポリシー定義の例:

{
    "mode": "All",
    "policyRule": {
        "if": {
            "allOf": [
                {
                    "field": "type",
                    "equals": "Microsoft.CognitiveServices/accounts/deployments"
                },
                {
                    "field": "Microsoft.CognitiveServices/accounts/deployments/sku.name",
                    "equals": "GlobalStandard"
                }
            ]
        },
        "then": {
            "effect": "deny"
        }
    }
}

ユースケース

UC1: 規制産業でのモデル使用制限

金融機関がデータ主権要件に基づき、Model Router のルーティング先を特定の承認済みモデルのみに制限。非承認モデルへのルーティング設定を含むデプロイメントを Deny で拒否。

-- コンプライアンス違反を事前に防止し、規制要件への準拠を自動化

UC2: コスト管理のためのデプロイメントタイプ制限

Provisioned (PTU) デプロイメントを特定のリソースグループに限定し、意図しないコスト発生を防止。GlobalStandard 以外の SKU タイプのデプロイメントを特定スコープで制限。

-- 予期しない高額なプロビジョニングを防止し、コストガバナンスを強化

UC3: セキュリティ承認済みモデルのみへのルーティング

本番環境では、セキュリティレビューを通過した承認済みモデルのみにルーティングする設定を強制。Audit 効果で非準拠のデプロイメントを検出し、レポートで可視化。

-- セキュリティ基準を満たすモデルのみが本番で使用されることを保証

料金

Azure Policy: 無料

Azure Policy 自体の利用は無料。Model Router / Foundry Models のデプロイメント料金はデプロイメントタイプに応じて課金されます。

Standard デプロイメント従量課金 (Pay-as-you-go)
