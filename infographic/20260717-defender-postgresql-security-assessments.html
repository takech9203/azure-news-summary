Microsoft Defender for Cloud Azure Database for PostgreSQL GA

Azure Database for PostgreSQL Flexible Server セキュリティ評価 一般提供開始

2026-07-17 | CSPM セキュリティ評価

概要

Microsoft Defender for Cloud の Cloud Security Posture Management (CSPM) による Azure Database for PostgreSQL Flexible Server 向けセキュリティ評価機能が一般提供 (GA) となった。PostgreSQL Flexible Server インスタンスのセキュリティ態勢を継続的に評価し、構成ミスの特定と優先度付きの推奨事項を受け取ることが可能。

CSPM は Defender for Cloud のコア機能であり、Microsoft Cloud Security Benchmark (MCSB) に基づくセキュリティ推奨事項を提供し、Secure Score と連携してセキュリティ態勢の改善を定量的に追跡できる。

Before

  • - PostgreSQL Flexible Server のセキュリティ構成を手動で確認する必要があり見落としが発生
  • - セキュリティ評価が体系化されていない
  • - 継続的なセキュリティ態勢の監視が標準化されていない
  • - コンプライアンス要件への適合状況を一元的に確認する手段が限られていた

After

  • - セキュリティ態勢が自動的かつ継続的に評価
  • - 優先度付きの推奨事項で最もリスクの高い構成ミスから対処可能
  • - Secure Score との連動でセキュリティ改善の進捗を定量的に把握
  • - Azure Policy との統合で組織全体にセキュリティポリシーを一貫して適用

主要機能

🛡

継続的セキュリティ評価

PostgreSQL Flexible Server のセキュリティ構成を自動的に評価。認証、暗号化、ネットワーク、監査などの領域を包括的にカバー。

📊

Secure Score 連携

推奨事項の対応状況が Secure Score に反映され、セキュリティ態勢の改善を定量的に追跡できる。

📋

優先度付き推奨事項

リスクに基づく優先順位で推奨事項を提示。最も重大な構成ミスから効率的に対処が可能。

📜

Azure Policy 統合

Microsoft Cloud Security Benchmark (MCSB) に基づくポリシー定義で組織全体に一貫したセキュリティポリシーを適用。

🔧

修復ガイダンス

各推奨事項に具体的な修復手順を提供。構成ミスの検出から修正までのワークフローを効率化。

対応プラン

プラン セキュリティ評価 追加機能
Foundational CSPM (無料) 基本的な推奨事項、Secure Score Asset Inventory
Defender CSPM (有料) 全推奨事項 + 攻撃パス分析、リスク優先順位付け Security Explorer、規制コンプライアンス

ユースケース

1. 金融業界のコンプライアンス対応

PCI DSS 要件に基づくデータベースセキュリティの継続的な準拠確認。暗号化設定やネットワーク分離の評価を自動化し、監査対応コストを削減。

2. 医療機関のデータ保護

HIPAA 要件に基づく PHI を格納する PostgreSQL の暗号化設定確認。転送中・保存時の暗号化、アクセス制御の適正性を継続的に検証。

3. SaaS プロバイダーのマルチテナント環境

複数 PostgreSQL インスタンスへのセキュリティポリシーの一貫した適用。Azure Policy との統合で全テナントに統一されたセキュリティベースラインを確保。

4. DevSecOps パイプライン統合

インフラデプロイ後のセキュリティ評価自動チェック。CI/CD パイプラインにセキュリティゲートを組み込み、構成ドリフトを早期検出。

設定方法

# Defender for Cloud の CSPM プラン状態を確認
az security pricing show --name CloudPosture

# セキュリティ評価の一覧を取得
az security assessment list \
  --query "[?contains(resourceDetails.id, 'Microsoft.DBforPostgreSQL')]"

# Defender CSPM を有効化
az security pricing create \
  --name CloudPosture \
  --tier Standard

制約事項

  • Foundational CSPM (無料) では基本的な推奨事項のみ
  • 実行時の脅威検出には別途 Defender for Open-Source Relational Databases プランが必要
  • 評価結果の反映にはタイムラグあり (数時間以内)
  • Single Server は対象外 (Flexible Server のみ)
