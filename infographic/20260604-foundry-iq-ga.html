Microsoft Foundry / Azure AI Search

Foundry IQ GA + 新 Knowledge Sources

マネージド知識レイヤーでエージェントのエンタープライズデータ活用を革新

GA - 一般提供開始 Preview - プレビュー

リリース日: 2026-06-04

📋概要

  • Foundry IQ はエンタープライズデータでエージェントをグラウンディングするためのマネージド知識レイヤー
  • SharePoint、OneLake、Azure Blob 等のソースを接続すると、インデックス・チャンキング・エンベディング・クエリ時応答を自動処理
  • Azure AI Search と Foundry Knowledge Base 間のプライベート接続が GA
  • 3 つの新 Knowledge Source がプレビュー: Fabric IQ Ontology、Azure SQL Database、Content Understanding

Before / After 比較

❌ Before（従来）

  • エージェントごとに検索パイプラインを個別構築（インデックス、チャンキング、エンベディング）
  • AI Search と Foundry 間はパブリックエンドポイントのみ（規制産業で問題）
  • Fabric セマンティックレイヤー (Ontology) への直接アクセス不可
  • Azure SQL を知識ソースとするにはカスタム ETL が必要
  • ドキュメント内の画像はテキストとして検索不能

✅ After（改善後）

  • マネージド Knowledge Base で即時マルチソース統合検索を実現
  • Private Link によるエンドツーエンド接続が GA（閉域ネットワーク対応）
  • Fabric IQ Ontology を Remote Knowledge Source としてセマンティックレイヤークエリ
  • Azure SQL をファーストクラスの Knowledge Source として自動インデクサーパイプライン
  • Content Understanding の画像言語化で画像をテキスト検索可能に

💡ユースケース

🏢 エンタープライズ知識エージェント

SharePoint、Fabric、Azure SQL を横断した統合検索エージェント。全社のナレッジを単一のインターフェースで検索し、グラウンディングされた回答を生成。Foundry IQ が複数ソースのインデックスを自動管理し、Agentic Retrieval でマルチステップクエリを最適化。

🏥 規制産業向けマルチモーダル文書検索

製薬研究における画像言語化 (Image Verbalization) を活用した文書検索。Content Understanding により、研究論文や臨床データの図表・画像がテキストとして検索可能に。Private Link GA によりデータが閉域ネットワーク内で完結し、HIPAA 等のコンプライアンス要件を満たす。
