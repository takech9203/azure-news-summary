Shortcuts Configuration & Results Grid 改善

Azure SQL Database Public Preview 2026-07-15

概要

Visual Studio Code 向け MSSQL 拡張機能に「Shortcuts Configuration (プレビュー)」が追加され、キーボードショートカットを GUI ベースで直接カスタマイズできるようになりました。Quick Queries による SQL スニペットのショートカット実行や、Beta Results Grid の列カスタマイズ機能により、Azure Data Studio からの移行がよりスムーズになります。

主要機能

⚙

Shortcuts Configuration Preview

MSSQL 拡張機能ツールバーから起動する GUI ベースのショートカット管理画面。Quick Queries と Extension Shortcuts の 2 タブ構成で、JSON ファイルの手動編集が不要に。

⚡

Quick Queries

SQL スニペットを複数保存し、キーボードショートカットで即時実行または展開。Auto-execute オプションにより、実行方法を柔軟に選択可能。

⌨

Extension Shortcuts

Query Editor (クエリ実行、接続) と Result View (ナビゲーション、結果操作) のショートカットをグループ別に表示・編集する機能。

▦

Beta Results Grid Preview

状態管理の改善、パフォーマンス向上、列のフリーズ・表示/非表示といったカスタマイズ機能を提供する新しい Results Grid。

🎯

MSSQL Database Management Keymap

SSMS や Azure Data Studio で使い慣れたショートカット (F5 でクエリ実行、Ctrl+N で新規クエリなど) を VS Code で使用できるコンパニオン拡張機能。

設定例

Beta Results Grid を有効にするには mssql.preview.betaResultsGrid を true に設定します。Result View ショートカットは settings.json の mssql.shortcuts キーで定義します。

JSON settings.json 
{
    "mssql.preview.betaResultsGrid": true,
    "mssql.shortcuts": {
        "event.queryResults.switchToResultsTab": "ctrl+alt+r",
        "event.queryResults.switchToMessagesTab": "ctrl+alt+y",
        "event.resultGrid.copySelection": "ctrlcmd+c",
        "event.resultGrid.selectAll": "ctrlcmd+a",
        "event.resultGrid.toggleSort": "alt+shift+o"
    }
}

ユースケース

DBA の日常運用効率化

データベースサイズ確認やインデックスチェックなど、毎日実行する定型クエリを Quick Queries に登録し、Ctrl+Alt+1 などのショートカットでワンキー実行。

毎日の定型作業がショートカット一つで完了し、手動でのクエリ入力・実行の手間を大幅に削減。

Azure Data Studio からの移行

Azure Data Studio 廃止 (2026年2月) に伴い、SSMS スタイルのショートカット (F5 で実行) に慣れたチームが VS Code へ移行するケース。

MSSQL Database Management Keymap と Extension Shortcuts の活用により、既存の操作習慣を維持しながらスムーズに移行完了。チーム全体の生産性低下を最小化。

注意事項

- Shortcuts Configuration は Public Preview であり、今後変更の可能性あり
- Beta Results Grid はデフォルト無効 (フラグ設定が必要)
- Keymap 拡張機能は VS Code デフォルトショートカットを一部上書きする可能性あり
- Azure Data Studio のカスタムキーバインドは自動移行されない (手動再設定が必要)
- Result View ショートカット (mssql.shortcuts) は VS Code の keybindings.json と別体系
