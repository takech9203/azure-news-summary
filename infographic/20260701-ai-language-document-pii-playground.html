Azure AI Language

Document PII Playground
(Microsoft Foundry NextGen)

ネイティブドキュメントからの個人情報検出・墨消しをコードなしで即座に評価

GA Public Preview 2026-07-01

概要

Azure AI Language の Document-based PII（個人情報検出）機能に、Microsoft Foundry NextGen ポータル上で利用可能な Playground エクスペリエンスが追加された。Public Preview と GA が同時発表され、PDF、Word (.docx)、テキスト (.txt) などのネイティブドキュメントファイルから個人情報を検出・墨消しする機能をコードなしで即座に評価可能になった。

Key Points

  • Microsoft Foundry NextGen ポータルに Document PII Playground を統合
  • プリロードされたサンプルドキュメントにより、準備不要で機能評価を開始可能
  • PDF、DOCX、TXT のネイティブドキュメントを直接処理し、構造・フォーマットを保持
  • GA と Preview の同時リリースにより本番環境と先進機能の両方をサポート
  • 墨消し結果のビジュアル表示によりステークホルダーへのデモが容易に

Before

  • Document-based PII 機能を試すには、Azure リソースの作成、ストレージの準備、API リクエストの構築が必要で、初期評価に時間がかかった
  • サンプルドキュメントが提供されていなかったため、テスト用ドキュメントをユーザー自身で準備する必要があった
  • 非同期 API のワークフローを理解するために開発者向けドキュメントを読み込む必要があった

After

  • Microsoft Foundry NextGen ポータル上で、コードなしで Document PII 機能を即座に試せるようになった
  • プリロードされたサンプルドキュメントにより、準備不要で機能評価を開始できる
  • 墨消し結果がビジュアルに表示されるため、ステークホルダーへの機能デモが容易になった
  • GA リリースにより本番環境からの Playground 利用が安定的にサポートされた

Playground サンプル体験

Microsoft Foundry NextGen ポータルに統合された Playground で、プリロードされたサンプルドキュメントを使用して即座に Document PII 機能をデモ・評価できる。コーディング不要でインタラクティブな体験を提供。

✅

リフレッシュされた GA Playground

キュレーションされたサンプル入出力により、コードなしで PII 検出機能を本番品質で評価可能。GA リリースにより安定した本番環境でのサポートを実現。

📄

Document-based PII の基盤機能

ネイティブドキュメント（PDF, DOCX, TXT）の直接処理に対応。ドキュメントの構造・フォーマットを保持したまま PII を検出・墨消し。設定可能なマスキングポリシーによる柔軟な制御。

Use Cases

🏦 コンプライアンス部門での評価

金融機関のコンプライアンス部門が顧客ドキュメントの PII 墨消し機能を導入検討する際に、Playground を使用してコードなしで即座に機能を評価。サンプルドキュメントで検出精度を確認し、本番導入の意思決定を加速。

効果: POC 期間を大幅に短縮し、ステークホルダーへのデモを即日実施可能に

⚖️ 法務ドキュメントの自動墨消し

裁判記録や契約書に含まれる個人情報（氏名、住所、電話番号など）を自動検出・墨消し。PDF の構造を保持したまま処理することで、法務部門のワークフローに直接統合可能。

効果: 手動での墨消し作業を自動化し、人的ミスによる情報漏洩リスクを低減

🤖 AI/ML パイプラインの前処理

AI モデルの学習データから事前に PII を除去し、プライバシーに配慮した学習データセットを構築。非同期 API により大量ドキュメントのバッチ処理に対応。

効果: GDPR/個人情報保護法に準拠した AI モデル開発パイプラインを実現
