# デフォルトテーマ

グラフィックレコーディング風のデフォルトテーマ。カラフルな配色と手書き風の要素を使用します。

## カラーパレット

```css
:root {
    --primary: #0078D4;
    --secondary: #50E6FF;
    --accent: #FFB900;
    --success: #107C10;
    --warning: #FF8C00;
    --error: #D83B01;
    --text-primary: #323130;
    --text-secondary: #605E5C;
    --surface: #FFFFFF;
    --surface-secondary: #FAF9F8;
    --border: #EDEBE9;
}
```

## フォント

```css
body {
    font-family: 'Noto Sans JP', 'Segoe UI', 'Meiryo', sans-serif;
}
```

## レイアウト

- シングルカラム
- 最大幅: 1000px
- レスポンシブ対応
